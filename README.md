# aros-tools

My AROS sandbox. Zune apps, utilities, and whatever else I end up 
building while learning to develop for AROS on x86_64 ABIv11.

Clone it, steal it, learn from it, do whatever. AROS fuckin' rocks.

## What's in here so far

- `zune-apps/FTPacket/` — minimal FTP client with Zune GUI (plain FTP 
  only, no SFTP/TLS, needed something to move files)
- `zune-apps/AROSQL/` — SQLite GUI client (work in progress)

## Status

Actively poking at this. Nothing here is production quality but it 
mostly works, which is a personal victory.
